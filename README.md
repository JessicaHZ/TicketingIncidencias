# Sistema de Ticketing de Incidencias (Proyecto Integrador)

Este proyecto implementa un **sistema de registro y gestión de incidencias** para el laboratorio LAB-LIS.  
Fue desarrollado como parte del **Proyecto Integrador**, aplicando principios de **Programación Orientada a Objetos (POO)** en Java.

Permite registrar incidencias, asignar técnicos, cambiar estados, listar y filtrar tickets, y generar reportes básicos.  
La interacción se realiza mediante un **menú por consola**.

---

# Características principales

- Registro de incidencias con:
  - Tipo (Hardware, Software o Red)
  - Prioridad (Alta, Media o Baja)
  - Equipo afectado
  - Descripción
- Asignación de técnicos
- Seguimiento del estado de incidencias:
  - PENDIENTE  
  - EN_PROCESO  
  - RESUELTA  
  - CERRADA  
- Historial de cambios
- Filtro por:
  - Estado
  - Tipo
  - Prioridad
- Generación de un reporte general
- Diseño basado en:
  - Clases
  - Herencia
  - Composición
  - Enumeraciones
  - Relación entre objetos
